## 📌 About This Project

This is a modern, responsive, and futuristic portfolio website designed with an AI-inspired UI.  
It represents my technical journey and highlights my work in:

- 🤖 Robotics & Automation
- 🧠 Artificial Intelligence
- 💻 Web Development
- ⚙️ Embedded Systems (Arduino, ESP32)

The design follows a **Tesla / Jarvis-style futuristic theme** with animations and interactive elements.

---

## ✨ Features

- 🌌 AI-inspired futuristic UI (glassmorphism + neon glow)
- ⚡ Animated particle background
- ✍️ Typing animation (dynamic role display)
- 📱 Fully responsive design (mobile + desktop)
- 💼 Projects section with images and live demo links
- 📄 CV download option
- 📬 Contact form (Formspree integration)
- 🎯 Smooth scroll + fade-in animations
- 🔍 SEO optimized (Google-friendly)

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (DOM Manipulation, Animations)
- tsParticles (Background animation)
- Formspree (Contact form backend)
